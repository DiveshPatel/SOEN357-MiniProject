Onboarding Process

The onboarding needed to be very quick, simple and intuitive process. The user has the option of logging in with their google or apple account. If the user has neither of these accounts, the option to register a new iFit account from scratch is always available. To make this process even faster, iFit allows the user to import their fitness data from a number of popular apps such as Samsung Health or Apple Health.

View and Add a Listing

This feature is the core of the application. Once logged in, this element will allow you to view your current listings and add new listings. Each listing provides information on the type of activity, the recorded duration of it and its estimated value for sale. The user can click on the + icon and to upload new data personally or import new data from other apps. The information that will be input will vary depending on its type.

The user can see their current account balance that they have earned by selling their data to buyers. Selective buyers that go through our screening process will be able to purchase this information that you share.
